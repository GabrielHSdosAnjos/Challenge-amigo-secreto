## 📌 Introdução

**Challenger Amigo Secreto** é um projeto desenvolvido como parte de um desafio proposto pela **Oracle** em parceria com a **Alura**. O objetivo principal era criar um script em JavaScript que permitisse listar os nomes dos seus amigos em um input e, ao clicar no botão de sorteio, selecionar aleatoriamente uma pessoa.

Decidi modificar bastante o projeto para deixá-lo com um design mais alinhado ao meu gosto pessoal. Como sou fã de videogames, me inspirei no estilo de **Red Dead Redemption 2** para dar uma estética diferenciada ao site.

## 🚀 Tecnologias Utilizadas

O projeto foi desenvolvido utilizando as seguintes tecnologias:

- **HTML5** para a estrutura do site.
- **CSS3** para estilização, incluindo adaptações para o design inspirado em *Red Dead Redemption 2*.
- **JavaScript (ES6+)** para a lógica do sorteio e manipulação dos elementos da interface.

## 🎮 Personalização

Além de implementar o sorteio de amigo secreto, fiz várias customizações para tornar o site mais envolvente, incluindo:

- Um layout que remete à estética do Velho Oeste, inspirado no *Red Dead Redemption 2*.
- Tipografia e cores temáticas para criar uma experiência visual imersiva.
- Efeitos sutis para tornar a interação mais dinâmica e agradável.
